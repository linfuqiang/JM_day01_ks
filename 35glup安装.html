1,装node（直接下一步，放C盘）——装好后 node -v 查看是否装好 npm -v
2,该镜像npm install cnpm -g --registry=https://registry.npm.taobao.org
3,cnpm -v 查看版本，如果出现乱码就再来一次 2步骤。
4,在cmd中执行 cnpm install gulp -g
5,gulp -v 查看版本，如果出现乱码就再来一次 4步骤。
6,当前项目根目录新建package.json文件
7,命令行进入根目录
8,cnpm init (初始化) 会填一些资料见 package.json 以后可以直接复制这个文件使用，稍作修改就可以了。
9,本地安装gulp插件，插件有很多。举例：安装less插件 都要在根目录的命令行
cnpm install gulp --save-dev
cnpm install gulp-less --save-dev
cnpm install gulp-clean-css --save-dev (压缩CSS)
cnpm install gulp-connect --save-dev (自动更新)
cnpm install gulp-imagemin --save-dev (图片压缩)
cnpm install gulp-uglify --save-dev (js压缩)
cnpm install gulp-rename --save-dev (修改文件名——可以不用)
开发依赖的插件，会在安装后配置在package.json
10,当前项目根目录新建gulpfile.js文件
11,配置好之后 调用 gulp + 自定义的命令
自动挡:
一：Less版
1,复制文件 到根目录
一、gulpfile.js (见Less环境文件夹)
二、package.json (见Less环境文件夹)
三、 node_modules (用命令 cnpm i 或者复制)
2,进入命令 然后进去根目录文件夹 gulp watch

二:完整版：
1、建立项目，删除所有文件，然后把最全环境里面的文件复制过来。(里面的gulpfile.js有两个版本，一般的开发板，root:dist,另外一个root:src)
2、 node_modules (用命令 cnpm i 或者复制)
3、gulp 根目录
如果出现错误,那错误的英文复制到搜索引擎中,找方法解决.
